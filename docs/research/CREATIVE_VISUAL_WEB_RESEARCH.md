## 1. 3D・WebGL型（世界の中を歩く）

### ◎ BRUNO SIMON PORTFOLIO（2019年）
- **ビジュアルメカニズム**: ポートフォリオ全体が俯瞰視点の3Dゲーム世界。ユーザーがRCカーを矢印キーで操縦し、地形内を移動して各プロジェクトゾーンに「到達」することがナビゲーション
- **体験**: カーがプロジェクトゾーンに入るとカメラがオーバーヘッドにドリー → スクロールでズームしてケーススタディを読む。看板を倒したり木に衝突したりできる物理演算あり
- **クラフトの高さ**: ポートフォリオ全体を1本のゲームエンジンで構築。草（78,400本の1ポリゴン葉）の物理シミュレーション、カスタム車両シェーダー、Cannon.jsによるリアルタイム物理演算
- **技術**: Three.js + Cannon.js + Blender + GLSL
- **受賞**: Awwwards Site of the Month 2019 / Awwwards SOTD / FWA
- **URL**: bruno-simon.com

### ◎ ACTIVE THEORY V6（Active Theory、2024年）
- **ビジュアルメカニズム**: ネオンライトで照らされた没入型3Dオフィス環境。グローする粒子トレイルがユーザーの動きに追従。プロジェクトは空間内の「部屋」に配置
- **URL**: activetheory.net
- **受賞**: Awwwards SOTD

### ◎ JAM3 FWA 100（Jam3、2022年）
- **ビジュアルメカニズム**: 3幕構成のWebGL没入型ナラティブ。「過去」=変形する岩石（描くと砕ける）、「現在」=スプリングジョイントで繋がった球体群（触れるとはじき飛ぶ）、「未来」=FWA100ロゴ形状の粒子群の中を飛行
- **クラフトの高さ**: カスタムガウスぼかし + 被写界深度シェーダー最適化。各チャプターが独自の物理演算
- **技術**: Three.js + WebGL + カスタムシェーダー（gaussian blur, bokeh DOF）
- **受賞**: Awwwards SOTD（7.92/10）/ CSS Design Awards Website of the Month
- **URL**: fwa100.jam3.com

### ◎ INFINITE PASSERELLA（Lusion / Edan Kwan + Marco Ludovico、2022年）
- **ビジュアルメカニズム**: 永続的に走り続けるリアルタイム3Dファッションランウェイ。決して止まらない
- **受賞**: Awwwards SOTD

### ◎ G. COLOMBEL PORTFOLIO 2024（Guillaume Colombel、2024年）
- **ビジュアルメカニズム**: ドラッグ方向に画像メッシュが物理的に変形・伸縮するWebGLカルーセル。リリース時に弾性的に元に戻る（スプリング効果）
- **クラフトの高さ**: 単純なドラッグスライダーを「素材の張力と弾性」として体験させる触覚的インタラクション
- **技術**: WebGL + Three.js + GLSL頂点シェーダー + Infinite Scroll
- **受賞**: Awwwards SOTD 2024（Creativity 7.47/10）
- **URL**: gcolombel.com

### ◎ ARISTIDE BENOIST PORTFOLIO 2021（Aristide Benoist + Jon Way、2021年）
- **ビジュアルメカニズム**: 28プロジェクトをスクロールなしで表示。サムネイルインデックス＋キーボードの数字キー（1〜28）で直接ジャンプ。各プロジェクトに固有のカラーシステムとレイアウト。WebGLレンダリングのホームページ画像
- **クラフトの高さ**: 28件の密度をページ遷移なしで優雅に処理する設計。「本を開くように」各プロジェクトが独立した視覚言語を持つ
- **技術**: WebGL, GLSL, JavaScript
- **受賞**: Awwwards Site of the Month 2021 / SOTD
- **URL**: aristidebenoist.com

### ◎ ACTIVE THEORY V5 / HAUS（Active Theory、2019年）
- **ビジュアルメカニズム**: 横スライドショーにリアルタイムWebGLカレイドスコープ（万華鏡）トランジションを適用。スライド切り替えのたびに万華鏡状の変形が挿入
- **URL**: activetheory.net
- **受賞**: Awwwards SOTD

### ◎ CLMT. PHOTOGRAPHY（Clément Merouani + Guillaume Colombel、2021年）
- **ビジュアルメカニズム**: 写真シリーズをブリスターパック（薬のPTPシート）として提示。各写真が「密封されたパッケージ」として佇んでいる。クリックすると「開封」アニメーションで展開
- **クラフトの高さ**: 写真展示に「製品パッケージング」の比喩を持ち込んだ独自のフレーミング

## 2. カーソル・ホバー連動型

### ◎ MAKEMEPULSE AGENCY WEBSITE（makemepulse、2022〜2023年）
- **ビジュアルメカニズム**: カーソルを動かすと過去プロジェクトの画像が軌跡として付いてくる「カーソルイメージトレイル」。エージェンシーの仕事の歴史がカーソルの動きに重なって現れる
- **URL**: makemepulse.com

### ◎ CUSP AGENCY PORTFOLIO（CUSP、2022年）
- **ビジュアルメカニズム**: 制作実績インデックスでカーソルを動かすと、その位置に合わせてプロジェクト画像がリアルタイムでディストーション（歪み）。カーソルが「レンズ」として機能
- **URL**: cuspsf.com

### ◎ POMARES PORTFOLIO 2024（Nicolas Pomares、2024年）
- **ビジュアルメカニズム**: タイポグラフィ一覧上でプロジェクト名にホバーすると、フルスクリーン表示で対応するプロジェクト画像が出現。テキストとビジュアルの瞬時の切り替え
- **受賞**: Awwwards SOTD 2024
- **URL**: pomares.fr

### ◎ WANDERGATES（UNIKO Studio、2024年）
- **ビジュアルメカニズム**: ホームページにカーソルトレイル×フローティング複数画像。プロジェクトページは視差付きイメージとシームレスなグローバル遷移

## 3. ドラッグ・無限キャンバス型

### ◎ UNSEEN STUDIO（Unseen Studio、2023年）
- **ビジュアルメカニズム**: 「World Page」——ポートフォリオ全体が無限の2DXYキャンバス上に散在。ドラッグで自由にパン移動し、任意の作品に近づいてクリック
- **クラフトの高さ**: ポートフォリオの「構造」をなくし、「発見」の体験を設計。バーチャル座標系で任意の方向に際限なく移動できる
- **受賞**: Awwwards SOTD February 2023 / Site of the Month
- **URL**: unseen.studio

### ◎ DAVIDE BARATTA 24（Davide Baratta / Unlearn Studio、2024年）
- **ビジュアルメカニズム**: 全プロジェクト画像を空間的に配置した2Dキャンバス。ドラッグで移動・スクロールでズーム。空間の密度と疎らさでプロジェクトの重要度が示される
- **URL**: davidebaratta.com
- **受賞**: Awwwards SOTD

### ◎ HELIOTROPY / STUDIO CRONICA（2023年）
- **ビジュアルメカニズム**: 自由ドラッグ型デスクトップキャンバス——写真・手書きメモ・動画クリップがランダム配置されたデスクトップUI。「誰かのデスクトップを見ている」体験
- ブランドのデジタルフォトブック設計として非常に独自性が高い

## 4. スクロール連動型

### ◎ QUECHUA 2025 LOOKBOOK（Quechua / Decathlon、2025年）
- **ビジュアルメカニズム**: 透視投影スクロールギャラリー——フルブリードのアウトドア写真がスクロールに合わせて「前方への移動感」で展開。登山体験を視点移動で体現
- **受賞**: Awwwards SOTD

### ◎ OBYS AGENCY（Obys Agency、2022〜2023年）
- **ビジュアルメカニズム**: カバー切り替えにアニメーションを伴うシネマティックスクロール。作品集を「映画のオープニングシーケンス」のように展開
- **URL**: obys.agency
- **受賞**: Awwwards SOTD

### ◎ TAKAMITSU MOTOYOSHI PHOTOGRAPHY（photoyoshi.com、2024年）
- **ビジュアルメカニズム**: デュアルモード無限スクロール＋水平横断。ホバーで全画面トリガー。写真家のポートフォリオを「ふたつの読み方」で体験できる
- **受賞**: Awwwards SOTD

### ◎ UNSEEN STUDIO（2023年別バージョン）
- **ビジュアルメカニズム**: カーソルパンで3Dジオラマ視差——ページ全体をカーソルで「ガラスケースの中のジオラマ」のように傾けて閲覧
- **受賞**: Awwwards SOTD

### ◎ CRISTÓBAL BALENCIAGA MUSEOA 10周年（2021年）
- **ビジュアルメカニズム**: 垂直スクロールを水平移動に変換——縦マウスホイールで横スクロール。コレクションの時系列を「タイムライン」として水平展開
- **受賞**: Awwwards SOTD

## 5. グリッド・レイアウト革新型

## 8. ナビゲーション設計の革新

### ◎ CUSP AGENCY（CUSP、2022年）
- **ビジュアルメカニズム**: 制作実績を「カーソルの位置でディストーションが変化する動的インデックス」として展開。通常のグリッドではなくカーソルが空間に干渉する体験

### ◎ CLMT. PHOTOGRAPHY（2021年）
- **ビジュアルメカニズム**: 写真をブリスターパック（薬のシート）として並べるパッケージングメタファー。「未開封のもの」という緊張感を写真展示に持ち込む

## ビジュアル表現の主要アプローチまとめ

| アプローチ | コア問い | 代表技術 |
|---|---|---|
| **3D世界を歩く** | ポートフォリオが「場所」なら、訪問者は旅行者になれるか？ | Three.js, Cannon.js, Blender |
| **カーソルに反応する** | 動きの軌跡自体がコンテンツになれるか？ | GSAP, canvas, WebGL |
| **無限キャンバスを漂う** | 「ページ」を捨てて「空間」にすると何が変わるか？ | Custom XY coordinate system |
| **スクロールで前進する** | 縦移動を「時間・空間の移動感」に変換できるか？ | GSAP ScrollTrigger, CSS |
| **グリッドを壊す** | グリッドの「整然さ」を意図的に破るとどうなるか？ | CSS Grid + GSAP |
| **変形・歪みで素材感を出す** | 画像が「デジタルなもの」ではなく「物理的な素材」に見えるか？ | WebGL vertex shaders |
